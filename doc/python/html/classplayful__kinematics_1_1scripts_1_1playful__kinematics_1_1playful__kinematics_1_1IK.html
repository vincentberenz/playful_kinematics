minimization_prioritydictionaty {joint name:integer score} the higher the priority (the lower the score) for a a joint, the higher priority it will have in the mimimization process if none, default is used (e.g. for Pepper, moving hips and knee is lower priority and the robot will avoid moving them, if possible)

Parameters

Returns
